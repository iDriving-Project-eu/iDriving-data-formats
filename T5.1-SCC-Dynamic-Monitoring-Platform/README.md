# T5.1 — SCC Dynamic Monitoring Platform

**Lead partner:** UNI.EIFFEL
**iDriving components:** IC23

Place the JSON files that describe the data formats exchanged by this component's
integration interfaces in this folder.

Naming convention: one file per interface, named by its interface ID as it appears
in the D5.4 deliverable, e.g. `T5.1-01.json`, `T5.1-02.json`.

See the [`_example/`](../_example) folder for a generic representative payload and the
header/topic conventions every message should follow.
